# NOT Gate in Verilog

## Overview

This project implements a **1-input NOT Gate (Inverter)** using **Verilog HDL** along with a testbench for simulation and verification. The design is compatible with **EDA Playground** and generates waveforms that can be viewed using **EPWave**.

## Project Files

* `not_gate.v` – Verilog implementation of the NOT gate.
* `not_gate_tb.v` – Testbench for simulating and verifying the NOT gate.
* `waveform.png` – Simulation waveform (optional).

## Features

* Simple and beginner-friendly Verilog implementation.
* Complete testbench for functional verification.
* Compatible with EDA Playground.
* Supports EPWave for waveform visualization.
* Ideal for learning digital logic design and Verilog HDL.

## Simulation

The testbench applies all possible input values to the NOT gate and verifies the output. A VCD file is generated during simulation, allowing the waveform to be viewed and analyzed using EPWave.
